About me
Who I am

Hi! My name is Sebastian. I'm a data scientist and software engineer originally from Singapore but currently based in Durham, North Carolina, where I just graduated from Duke University with a B.S. in Math this May.

I have been coding for about three years now and found the greatest joy in building data-driven products powered by artificial intelligence, in particular deep learning, computer vision and natural language processing. I also enjoyed building web applications that integrate with data products to bring the power of big data to the fingertips to thousands and even millions of people and provide them with exquisite user experiences. Check out some of my skills and projects below!

While I'm not coding and building on my next idea, I enjoy learning foreign languages on Duolingo, designing infographics, playing my violin, and playing and creating mods for Civilization - my favorite game.

I am always looking to connect, learn and grow with more awesome people working in all fields of tech. Please don't hesitate to reach out to me through any of the following channels. I would love to learn about you and yor work!

Projects
What I have done


Restaurant Dish Recommendation Engine

Technologies: AWS, Azure, Caffe, Python

For this project, I

  • Prototyped recommendation algorithms to suggest dishes based on user location and taste using Keras, Azure and AWS machine learning suites and achieved 0.7+ Spearman correlation

  • Deployed the best scoring algorithm as a scalable prediction API to AWS using S3 and ECS

  • Created a MongoDB database of dish names, descriptions and taste levels using data from Locu and Yummly APIs for 100K+ restaurants in the US and the UK

Building Energy Data Analytics

Technologies: C++, ARCGIS, Caffe, MATLAB, Python

For this project, I

  • Developed a pixel-based computer vision algorithms to detect buildings in satellite images with 0.86+ AUC score using MATLAB for image preprocessing, C++ for image segmentation and Caffe for deep learning

  • Created and open-sourced a ground-truth dataset of satellite images, structural footprints and heights of 40K+ buildings in 7 US cities and suburbs using ARCGIS, OpenStreetMap and LIDAR


Predicting and Visualizing Vacation Choices for Expedia Users

Technologies: D3.js, Google Maps, Gephi, Leaflet.js, Python, R, Tableau

For this project, I

  • Visualized a network of 1M+ travel origins and destinations searches by Expedia users using Gephi, Leaflet.js with Google Maps layers, and Tableau

  • Performed user segmentation on 100K+ users through K-means clustering in R and extracted hotel choice insights based on date, travel distance and group size



Post-2016 Election Sentiment Analysis

Technologies: Google BigQuery, R, GDELT , Tableau

For this project, I

  • Constructed and optimized SQL queries using Google BigQuery to extract data from the 2.5 TB+ global knowledge graph from Global Database of Events, Languages and Tones (GDELT)

  • Created sentiment time series analysis of news coverage of the 2016 Elections and its outcome using Tableau visualizations


Predicting Housing Prices in Ames, IO

Technologies: Azure, D3.js, Python, R

For this project, I

  • Created interactive visualizations using D3.js and Tableau for housing data in Ames, Iowa

  • Built prototypes of prediction algorithms using Scikit-Learn through feature engineering, gradient boosting and ensemble learning and achieved 0.72+ cross validated R2


Modelling Chaos in Physics

Technologies: Python, Mathematica

For this project, I

  • Built a proof of concept circuit capable of detecting muon to electron (Mu2e) conversions currently used in Beyond Standard Model physics research at Fermilab

  • Modeled chaotic dynamics of double pendulum using Python
